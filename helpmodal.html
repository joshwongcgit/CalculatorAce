History

History section

The history section keeps track of your past calculations. This will enable users to have an easier and efficient way of solving long mathematical problems

Using history to reselect

You can click on any elements to add to your current calculation. You can actually visualize previous results so you input them into your next problem talk about result section

Color indicators

Hovering on elements in the history section creates colorful indexes that helps you distinguish the different kinds of elements. All types of elements exibit different colors.

Managing equations

Clear history

The past calculation results will be cleared. If there is a calculation in process, it will remain and be continued at the top.

Clear calculation

Clears only the calculation section. The next caculation will form a new line in the history section.

Clear all

Deletes everything from the calculation to the history section. Useful if you want to start a long separate problem

Add tab

Use the O key to add a new tab while staying on the current tab. This will distinguish different sets of equations.

And add and start at new tab

Use the K key to open and begin your next set of assignment in a new tab.

Delete tab

Use the P key to delete the current tab.

Rename tab

This program offers a feature where you can customized the tab names. It will organize your equations and you can go back to it easier. This is done by double clicking on the tab that you would want to rename

Next tab

Use the tab key to focus on the next tab, if any.

Previous tab

Use the esc key to focus on the previous tab, if any.

Other Helpful features

Console messages

Console message appear below the instant answer section when you made an error in your calculation The message displayed will alert you the type of error. All events will be stopped and the incorrect value will not be inserted into your current calculation. This would prevent calculation error after even after sleecting equal sign. The console message visibility button allows you to either show or hide the console message.

Instant answer

The instant answer visibility button lets you hide or show the instant answer. Your answers are that quick! Once an equation
is typed out, the answers are displayed relatively below the current calculation.

Settings

The setting allows you to configure the program. Options can be changed using hotkeys

Hotkeys

Click on the hotkeys button to take a look. Certain mathematical values can be typed using the keyboard.

Single and whole digits

The numeric values can be reseelected in the history section as separate digits or as a whole. Reclicking the single or whole digit converter toggles it.
